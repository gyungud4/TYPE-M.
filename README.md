## 🎯 1. 기획 의도
"실무 중심의 **일반 전공**인가, 대회 중심의 **기능반**인가?"
입학 후 진로 선택에 고민이 많은 특성화고 1학년 신입생들을 위해, 성향 테스트를 바탕으로 최적의 세부 전공과 공부 팁을 추천해 주는 웹 서비스입니다.

---

## ✨ 2. 핵심 기능
* **🧩 2단계 진로 분석 알고리즘:** 
* **STEP 1:** 일반 전공군(gen) vs 기능반(skl) 성향 판별 (12문항)
* **STEP 2:** 내 성향에 맞는 14개 세부 분야 중 최적의 학과 도출 (20문항)
* **💬 선배들의 한마디:** 추천 전공별 핵심 기술 가이드 및 현실적인 조언 제공

---

## 🛠 3. 기술 스택 (Tech Stack)
* **Frontend:** HTML5, CSS3
* **Backend:**  Vanilla JavaScript 
* **Database/Auth:** LocalStorage
